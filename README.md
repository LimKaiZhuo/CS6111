# CS6111
CS611 Labs and Assignments
Github Link:
https://github.com/LimKaiZhuo/CS6111/tree/main/assignment_1

To run:
1) clone repo
2) Make sure Docker Desktop is running.
3) Open a terminal and navigate to the `assignment` directory inside the cloned repository.
   cd CS6111/assignment
4) Run the command:
   docker-compose up
5) Open the Jupyter Notebook URL that appears in your terminal (it will look something like http://127.0.0.1:8888/lab?token=...).
6) Inside Jupyter, open a new terminal.
7) Run the main script:
   python main.py
